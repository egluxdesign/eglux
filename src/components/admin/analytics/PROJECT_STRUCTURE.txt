FEATURES SUMMARY
═══════════════════════════════════════════════════════════════

✅ AUTHENTICATION
   • Supabase auth (email/password)
   • Session persistence
   • Logout functionality

✅ DASHBOARD (7 Menus)
   1. Dashboard     - KPI cards + recent orders
   2. Orders        - Full CRUD + filter + sort + pagination
   3. Products      - CRUD + variants + stock management
   4. Customers     - List + detail + order history
   5. Payments      - Track transactions (Midtrans-ready)
   6. Analytics     - 3 chart types + date range
   7. Settings      - Store info + Midtrans credentials

✅ ORDERS MANAGEMENT
   • Search by customer/phone/ID
   • Filter by status + date range
   • Sort by all columns
   • Pagination (10/25/50/100)
   • Bulk status update
   • Export CSV
   • Detail modal with items

✅ PRODUCTS CRUD
   • Image preview
   • Variant manager (size, color, price, stock)
   • Low stock alert
   • Bulk actions
   • Delete confirmation

✅ CUSTOMERS
   • Order count + total spent
   • Last order date
   • Detail modal with full history
   • Stats cards

✅ PAYMENTS
   • Status: Pending, Paid, Failed, Expired, Refunded
   • Transaction ID tracking
   • Revenue stats
   • Midtrans integration ready

✅ ANALYTICS
   • Revenue trend (Area/Line chart)
   • Top products (Bar chart)
   • Status distribution (Pie chart)
   • Date range filter
   • 4 KPI cards + 3 metric cards

✅ DESIGN
   • Dark sidebar + light content
   • Eglux gold (#c9a96e) accent
   • Responsive (mobile sidebar)
   • Skeleton loading states
   • Empty states
   • Smooth transitions
